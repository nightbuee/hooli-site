API Documentación

Introducción

Hooli es una API JSON basa en REST. Proporciona funcionalidad para un sitio de venta de cartas coleccionables por medio de solicitudes de HTTP. Esta documentación describe todas las acciones que puedes realizar así como sus parámetros, envíos, salidas y métodos de autentificación.

Autentificacion

Para poder hacer uso de la API de Hooli necesitas tener un token, la autentificación es Key Based por lo tanto al momento de hacer las peticiones con los verbos HTTP tienes que enviar el token dentro del header en la petición en una variable llamada Authorization como se muestra a continuación.

Header
key value
Authorization* EyJ0eXAiOiJKV1QiLCchbGciOsJIUzI1NiJ9.eyJ1IjoiNWYyOGM5ODRmMzU4NmM0MTI4NDRkMmIyIiwic3ViIjoiZGlhbmEyIiwiaWF0sjoxNTk2N TA4NjgxLCJleHAiOjE1OTcxMTM0ODEsImVtYSI6ImRpYW5hMkBob3RtYWlsLmNvbSJ9.Dd49vrweygUaA1jig8WgWMKcZ8XjLlnt3f-jXRcEBjc

Las key con * son obligatorias.


POST Registro

Para poder empezar a utilizar nuestra API primero necesitamos obtener nuestro token de acceso y solamente necesitamos registrarnos.

url

Body
key value
email* test@hotmail.com
displayName* test
password* thisismypassword

Después de realizar el registro la API nos responderá con nuestro token.

Las key con * son obligatorias.

NOTA: Cuando te registras recibes una bonificación de 50US.

POST Login

Los token expiran en un periodo de 7 días por lo que si expira tu token o no lo anotaste puedes generar un nuevo token lo único que necesitas es iniciar sesion.

url

Body
key value
email* test@hotmail.com
password* thisismypassword

Si tus datos son correctos la API te responderá con un nuevo token .

Las key con * son obligatorias .

DELETE Eliminar carta

En caso de querer retirar una tarjeta de Hooli se puede realizar por este metodo.

url

Header
key value
Authorization* EyJ0eXAiOiJKV1QiLCchbGciOsJIUzI1NiJ9.eyJ1IjoiNWYyOGM5ODRmMzU4NmM0MTI4NDRkMmIyIiwic3ViIjoiZGlhbmEyIiwiaWF0sjoxNTk2N TA4NjgxLCJleHAiOjE1OTcxMTM0ODEsImVtYSI6ImRpYW5hMkBob3RtYWlsLmNvbSJ9.Dd49vrweygUaA1jig8WgWMKcZ8XjLlnt3f-jXRcEBjc
Body
key value
validate* test

Las key con * son obligatorias.

No se puede eliminar cartas de otros usuarios.

NOTA: Para poder eliminar la carta tenemos que enviar por body la key 'validate' y como valor nuestro nombre de usuario. En caso contrario no se eliminara la carta.
